# My Floating Stopwatch

## What is this?
This is a small stopwatch app that floats on top of all other windows.
- It stays on top even if you watch a movie in full screen.
- You can drag it anywhere on your screen.
- It has a simple dark look.

## How to install

This app works on **Windows** and **macOS**. 

1.  Make sure you have **Node.js** installed on your computer.
2.  Open your folder in the terminal.
3.  Run this command to build the app:
    ```bash
    npm run make
    ```

### For Windows users
- Go to the folder `out/make/squirrel.windows/x64/`.
- Open the `.exe` file (like `my-stopwatch-...-Setup.exe`) to install.
- A shortcut will be added to your Desktop and Start menu.

### For Mac users
- Go to the folder `out/make/zip/darwin/x64/`.
- Open the `.zip` file to get the App.
- Drag the app to your **Applications** folder.
